## Keyboard Shortcuts

### Play Notes
```
_WE_TYU_OP_
ASDFGHJKL;'
```

### Change Octave
`Z` / `X`

### Select Fader

`Tab` / `Shift`+`Tab`

### Move Fader

`Up` / `Down`

## Development

```shell
yarn install
yarn start
```

Then open [localhost:7200](http://localhost:7200/)

## Testing

via Jest

```shell
yarn test
```
